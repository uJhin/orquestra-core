.. py:function:: random_interaction_operator_term(order: int, real: bool = True, seed: Optional[int] = None) -> orquestra.quantum.openfermion.InteractionOperator

   Generates a random interaction operator with non-zero coefficients only
   on terms corresponding to the given number of unique orbitals.

   The number of orbitals is equal to the given order.

   :param order: How many unique orbitals the non-zero terms should correspond to.
   :param real: Whether or not the coefficients should be real. Defaults to True.
   :param seed: The seed. If None (default), uses np.random.

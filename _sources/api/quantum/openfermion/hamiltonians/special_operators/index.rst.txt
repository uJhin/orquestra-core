.. py:function:: number_operator(n_modes: int, mode: Optional[int] = None, coefficient=1.0, parity: int = -1) -> orquestra.quantum.openfermion.ops.operators.FermionOperator

   Return a fermionic number operator.

   :param n_modes: The number of modes in the system.
   :type n_modes: int
   :param mode: The mode on which to return the number
                operator. If None, return total number operator on all sites.
   :type mode: int, optional
   :param coefficient: The coefficient of the term.
   :type coefficient: float
   :param parity: Returns the fermionic number operator
                  if parity=-1 (default),
                  and returns the bosonic number operator
                  if parity=1.
   :type parity: int

   :returns: operator (FermionOperator)
